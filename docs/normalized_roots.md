## Notes

- Standardized headwords to dictionary lemmas (for example, **volvō**, **iungō**, **legō**).
- Separated the **pendō / pendeō** overlap so **pens-** is explained cleanly.
- Reframed **archon** under the root family **archē / archein**.
- Grouped the **spec- / spect-** family for practical English derivational use.
